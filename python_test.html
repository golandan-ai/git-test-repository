Python Backend Test

Purpose: Demonstrate that HTML can call a Python function on Vercel.
API Endpoint: /api/hello?text=your_text
Python processes: reverse, uppercase, lowercase, char count, word count, etc.

Enter text for Python to process:


How it works:

  1. HTML sends text to /api/hello?text=...
  2. Vercel routes request to api/hello.py
  3. Python processes the text (reverse, count, etc.)
  4. Python returns JSON response
  5. HTML displays the result

Future use: This same pattern can process Liaoshi data - clean input, translate, analyze, format output, etc.
